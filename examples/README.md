# Examples

This directory contains example integrations to help you get started.

## Available Examples

### 1. `simple-module.json`
A basic module that shows:
- How to define parameters
- How to make an API call
- How to use conditions
- How to map the response

**Use this example when**: You're creating your first module

### More examples coming soon:

- Invoice processing with sequential queue
- Customer support ticket classification
- Data synchronization between systems
- Webhook handling

## How to Use Examples

1. Read the JSON file to understand the structure
2. Copy the example to `/src/modules/`
3. Modify it for your specific use case
4. Test in Make.com platform
5. Adjust based on results

## Learning Path

Start with these examples in order:

1. **simple-module.json** - Basic structure
2. **(Coming soon) api-connection.json** - How to authenticate
3. **(Coming soon) error-handling.json** - Handle errors properly
4. **(Coming soon) sequential-processor.json** - Process items in order
5. **(Coming soon) webhook-receiver.json** - Receive external events
